# landing-odigo
Adaptive landing page for the Odigo travel service, featuring a clean modern design, structured layout, and a strong focus on visual content.

Demo: https://kod1sdev.github.io/landing-odigo/

Features:

* Fully responsive layout (mobile / tablet / desktop)
* Modern minimalist UI
* Semantic and clean HTML markup
* Optimized images
* Well-structured project architecture
* Smooth animations (if applicable)
* Cross-browser compatibility

Stack:

* HTML5
* CSS3 / SCSS
* Flexbox / Grid
* BEM methodology
